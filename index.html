Notable Projects

I learn the best when I apply the concepts and theories I have studied, into projects! I always make sure to incorporate past topics I have learned, along with new concepts, to new projects. This ensures that I don't forget them. I also take good practices to heart, especially ensuring I make good commit messages and comments for my future self.

And so, here are a compilation of projects that I'm particularly proud of. If you want to see the rest of the projects that I've created/currently working on, please check my github account.

The following projects, except 2D Targetting Practice and City Habit Tracker (group project), are made outside of university classes, in my own free time.

Rock Paper Scissors

A rock paper scissors game set in the browser. Inspired from pokemon battles, this game show to users the amount of turns they have left through life points.

Etch-a-Sketch

A canvas program set in the browser. Users can set the canvas size, choose different colour palettes, change ink, erase their drawing, eye drop existing colours, and toggle the canvas grid.

Calculator

A browser calculator that enable users to perform addition, subtraction, division, multiplication operations.

Sign Up Page

This project shows a mock sign up page for a non-existent online store. It involves run-time form validation to ensure users input the correct data.

2D Targetting Practice

(This project is currently still in progress. It is 70% completed.)

A JavaFX application that create targets for the user to resize, delete, move. After clicking the targetes, a graph of their clicking speed is presented to the user.

City Habit Tracker

(This project is set to private. If you want to explore, please contact me through email to ask for the project build.)

This project is made with JavaFX as its frontend and AWS as its backend. It enable users to create a city based from tasks and goals that they have accomplished.
